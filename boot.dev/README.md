# Boot.dev — Backend Path (Python/Go)

My tracked course list from the [Boot.dev Backend Developer Path](https://www.boot.dev/paths/backend?tech=python-golang), aligned to my SDE / backend goal.

> **Pricing reality:** Boot.dev has **no per-course paywall**. Every course is **free to READ** in the browser. A **membership** unlocks the interactive coding sandbox (run/submit code, boss battles, achievements, certificate, progress tracking). So the split below is **read-free** vs **needs-membership-to-*do***, not free-course vs paid-course.

---

## 🟢 Free (read in browser, no login needed)

Every lesson on this page is readable for free — theory, examples, explanations. Good for skimming concepts without paying.

- All 23 courses below are **readable free**. Membership only gates the *hands-on* practice.

## 🔵 Needs Membership (to actually code/submit/get cert)

Same 23 courses — running exercises, boss battles, and the certificate require a paid membership.
